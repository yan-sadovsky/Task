# Playwright Typescript

### **Project setup**

1. create `.env` file if not exists (example in [env.test](env.test))

```
cp env.test .env
```

2. run setup (require rimraf: npm install --global rimraf)

```
yarn start
```

### **Run tests**

```
yarn tests
yarn tests:headed
```

### **Show allure report**

```
yarn allure:report
```
